# math_629_final
LOB Price Movement Prediction with PCA and Market Factors

This project aims to answer the following questions:
Can deep learning models such as CNN, LSTM, and Transformer effectively extract predictive signals from cryptocurrency limit order book data?


Does PCA-based factor compression improve model stability and predictive accuracy by reducing noise and multicollinearity in LOB features?


Do additional market state variables, including volatility and trend indicators, improve short-term price movement prediction?


Can the predicted signals generate economically meaningful trading performance after accounting for transaction costs?
